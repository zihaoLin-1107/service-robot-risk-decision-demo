# GitHub Upload Plan

1. Create a new GitHub repository for the public demo only.
2. Push `public_demo_repo/` only.
3. Keep the full implementation private.
4. Commit `media/case_a_demo_video_compressed.mp4` directly with the public demo repository.
5. Optionally upload a higher-quality local video source to a GitHub Release or an external video platform.
6. If an external video URL is used, add it to `README.md` and `demo_video_link.txt`.
6. Do not add an open-source license.
7. Share the full private repository only with academic reviewers if needed.

## Pre-upload Checklist

- Confirm no raw logs, raw labels, private drafting materials, contact lists, or outreach drafts are included.
- Confirm no full source tree is included.
- Confirm only the compressed `.mp4` is committed and larger local video sources remain outside the public repository.
- Confirm `NOTICE.md` and `NO_OPEN_SOURCE_LICENSE.md` are present.
