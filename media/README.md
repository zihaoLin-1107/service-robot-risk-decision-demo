# Media Assets

This folder contains public-facing visual assets for the demo repository.

## Files

- `demo_thumbnail.png`: thumbnail image for the Case A demo video.
- `case_a_demo_video_compressed.mp4`: compressed demo video intended for the public repository.
- `decision_sandbox.png`: public visual explaining the decision-sandbox/pipeline stage.
- `webots_case_a.png`: Webots Case A screenshot with sensor-view overlay.
- `rviz_marker_path.png`: public visual for the RViz/branch-validation stage.

## Notes

These images are intended for public project overview and demo explanation. They are not raw experiment logs and do not include private drafting history, label tables, or full source code.

The compressed `.mp4` can be committed directly because it is small. If a higher-quality video source is hosted externally, add the video URL to the root `README.md` and `demo_video_link.txt`.

Do not commit large local video files directly to git. Upload them through a GitHub Release or an external video platform if needed.
