# PhD Application Demo v0.1

This release contains the public demonstration materials for a PhD application research prototype on human-calibrated risk-sensitive decision making for non-intrusive assistive mobile manipulation.

## Included

- Public README
- English and Chinese project briefs
- Case A Webots/RViz branch mapping demo video
- Demo screenshots
- Summary result files
- Public research claims
- Current limitations
- No-open-source-license notice

## What This Release Demonstrates

- A factorized decision context: `user_state + event_state + risk_state`
- A risk-sensitive behavior-tree decision layer
- A hard safety override mechanism
- Preliminary human preference calibration
- Webots/RViz `selected_action`-to-execution branch mapping

## What This Release Does Not Claim

- Not a complete service robot product
- Not a formal user study
- Not RL/PPO training
- Not full online perception-conditioned deployment
- Not real grasping or MoveIt manipulation

## Full Implementation

The full implementation, raw experiment logs, raw human labels, prompts, SOP materials, and advisor strategy documents are kept private and can be shared with advisors upon request.
